# dotfiles
Personal Dotfiles

## About
This contains my dotfiles and productivity software links.

## Installation
1. Install Vim and Vundle
2. Install ZShell
3. `make`
4. Install solarized color scheme
5. Install [Powerline Fonts](https://github.com/powerline/fonts) and change terminal font
