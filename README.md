# dotfiles
Personal Dotfiles

## Installation
1. Install Vim and Vundle
2. Install ZShell
3. Copy the maximum-awesome repo's bundles
4. Run `:PluginInstall` in Vim
5. Install solarized color scheme
6. Install [Powerline Fonts](https://github.com/powerline/fonts)
